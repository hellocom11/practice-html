메인페이지 이동 절대경로와 상대경로 부분으로 이동

속성

태그에 부가적인 내용(보이는 컨텐츠를 제외한)을 입력할 때 사용하는 문법

속성은 시작태그에 한칸 띄어서 입력한다.

속성표현법 - [속성명="속성값"]

속성은 모든태그에서 사용가능한 글로벌 속성과 특정태그에서만 사용할 수 있는 로컬속성으로 나뉜다

태그에 속성은 2개 이상 여러개를 입력할 수 있으며, 순서는 상관없다.

링크태그

우리가 브라우저의 주소창에 www.naver.com쓰고 엔터치면 네이버서버에게 메인페이지를 보여달라는 요청을 하는 것과 같이, a태그를 클릭하면 a태그에 href속성에 적용되어있는 절대경로 및 상대경로의 요청을 수행하는 것과 같다.

사용자가 링크태그를 클릭했을 때 href속성에 저장된 경로로 페이지를 이동시켜주는 태그

a태그는 href속성이 반드시 기재가 되어있어야 함

a태그는 target속성에 _blank값을 적용하여 새 탭에서 열게할 수 있다.(기본값:_self)

a태그는 문맥적인 뜻이 없으며, 단순히 링크를 만들 때 사용하는 태그 -> 강조태그처럼 사용
