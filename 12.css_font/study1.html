font

  • font-size -> 글씨의 크기를 지정하는 속성

    브라우저의 글씨크기 기본값은 16px이다.

    글씨의 가장 높은 부분에서 낮은 부분까지의 길이를 기준으로 한다.

    1em-> 부모태그에 적용된 글씨크기

    1rem-> html태그에 적용된 글씨크기

  • line-height -> 행간(글씨를 감싸는 보이지 않는 라인의 높이)

    행간은 기본적으로 120%로 설정되어있다. 120%는 1.2라고 소수로 써도 된다. line-height로 설정된 텍스트 공간 안에서 글자는 세로기준 중앙에 배치되는 특징을 갖는다.

  • font-weight-> 글씨의 굵기 설정

    글씨들마다 지원하는 굵기의 범위가 다르기 때문에 이를 잘 확인하여 사용

    • bold - 두껍게
    • normal - 일반(기본값)
    • lighter - 얇게
    • 100~900단위의 숫자를 사용해서 지정가능 - 숫자가 커질수록 폰트가 굵어짐(기본값400)

  • white-space-> 텍스트를 강제로 한줄로 나열되도록 설정
